# 职业性格测评系统

一个专业的MBTI和霍兰德职业兴趣综合测试网站。

## 功能特点

- 🧠 **完整的MBTI测试** - 50道题目
- 🔧 **霍兰德职业测试** - 60道题目  
- 📊 **可视化分析报告** - 图表展示
- 💼 **个性化职业推荐**
- 📱 **响应式设计** - 支持移动设备

## 在线访问

访问地址：https://你的用户名.github.io/career-assessment

## 技术栈

- HTML5 / CSS3 / JavaScript
- Chart.js 数据可视化
- Font Awesome 图标
- GitHub Pages 部署

## 本地运行

1. 克隆仓库
2. 用浏览器打开 `index.html`
